* web browser를 통해 kibana에 접속한다
    * local에서 사용하는 경우 : http://localhost:5601
    * cloud에서 사용하는 경우 : http://{ip주소}:5601
        * {}를 제거하고 ip를 입력한다.
        * 예) http://12.345.6.789:5601

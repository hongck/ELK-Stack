### 사전 준비

<a name='connect'></a>
#### web browser를 통해 kibana에 접속한다

* local에서 사용하는 경우 : http://localhost:5601
* cloud에서 사용하는 경우 : http://{ip주소}:5601
    * {}를 제거하고 ip를 입력한다.
    * 예) http://12.345.6.789:5601

<a name='discover'></a>
#### Side Navigation에서 `Discover`를 선택한다.

* 그러면 default index로 선택된 data가 표시된다.
